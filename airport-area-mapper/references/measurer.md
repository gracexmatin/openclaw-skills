# Airport Measurer 使用说明

交互式机场建筑物面积测量工具。

## 功能

- **标定模式**: 选择两个点，输入实际距离，建立像素-米比例
- **测量模式**: 框选建筑物，自动计算占地面积
- **导出**: 带标注的结果图 + CSV数据表

## 使用方法

```bash
python3 airport_measurer.py <图片路径>
```

## 操作流程

### 1. 启动程序
```bash
python3 airport_measurer.py airport.png
```

### 2. 标定比例尺 (按 C)
- 点击两个已知实际距离的点（如跑道两端）
- 输入实际距离（米）
- 程序自动计算 `1像素 = ?米` 的比例

### 3. 测量建筑物 (按 M)
- **左键点击**: 添加多边形顶点
- **中键点击**: 撤销上一个点
- **右键点击**: 完成测量
- 输入建筑物名称和备注

### 4. 保存结果 (按 S)
- `*_measured_*.png` - 带标注的图片
- `*_measurements_*.csv` - 数据表格

### 5. 退出 (按 Q)

## 快捷键

| 按键 | 功能 |
|------|------|
| C | 进入标定模式 |
| M | 进入测量模式 |
| S | 保存结果 |
| R | 重置显示 |
| Q | 退出程序 |

## 输出示例

```
=== 测量汇总 ===
总计: 3个建筑物, 125000.00平方米 (12.5000公顷)
  - T1航站楼: 45000.00m | 4.5000ha | 主航站楼
  - 停机坪A: 60000.00m | 6.0000ha | 客机停机
  - 货运区: 20000.00m | 2.0000ha | 货运处理
```

## 注意事项

- 需要图形界面环境（X11 或 Wayland）
- 图片分辨率越高，测量越精确
- 标定时选择水平/垂直方向的参考物，减少透视误差
